WHAT'S NEW IN 0.94.5

Release Notes

0.94.5 makes the game feel sharper, stranger, and more alive with new progression hooks, stronger weapon identity, and a fuller event layer.

  • Added the Boss Codex / Encounter Log to Records.
  • Added live announcements, the release notes popup, and a live player count in the main menu.
  • Added the St. Patrick's event preview, Lucky Theme unlock, and theme selector.
  • Added weapon masteries at 5 stacks for Freeze Ray, Black Hole, Shuriken, Homing Missiles, and Accelerator.
  • Reworked Freeze Ray, Black Hole, Laser Beam, Shuriken, Soul Reaver, and Hazmat-related combat polish.
  • Expanded the admin panel and separated the admin build into its own save database.
